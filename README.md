# 水産業 学習・評価アプリ

小学5年社会科「水産業のさかんな地域」のための、2画面構成の学習支援アプリです。

- `/hyouka`：第1〜7時のまとめを、授業ごとの規準でA・B・C判定
- `/sakusen`：「水産業レスキュー作戦」を6観点で自己点検

標準の「無料ルーブリック判定」はブラウザ内で動き、外部APIも料金も使いません。任意でGPT-5.4 nanoによる精査を有効にできます。

入力した児童のまとめ本文は保存しません。児童名、出席番号、学校名など、個人を特定できる情報は入力しない運用を前提にしています。

## 判定の考え方

- B：本時の目標をおおむね達成
- A：Bに加え、関連付け、意味、立場の多面性が質的に深まっている
- C：資料の根拠、複数の事実、関係付けのいずれかに教師の支援が必要

文章だけを見た補助結果なので、教師がノート、発言、学習過程を合わせて最終判断してください。

## GPT-5.4 nano精査の安全設計

AI精査はサーバー側の `/api/evaluate` からOpenAI Responses APIを呼び出します。APIキーをブラウザへ送信しません。

この版では、次の対策を組み込んでいます。

- モデルを `gpt-5.4-nano` に固定
- AI出力を最大350トークンに制限
- 児童の入力を最大1,200文字に制限
- 月額予算に達する前にアプリ側でAPI呼び出しを停止
- 月間リクエスト回数でも停止
- 同じ端末・回線からの連続送信を1分単位で制限
- 任意の「クラス用合言葉」で公開URLの第三者利用を防止
- API失敗時や上限到達時は無料ルーブリック判定へ自動切替
- D1には利用回数、トークン数、概算料金、ハッシュ化した制限用識別子だけを保存
- 児童が入力したまとめ本文はD1へ保存しない

> OpenAI側のプロジェクト予算は通知用です。このアプリはD1に記録した実利用額と回数を基に、APIを呼ぶ前に独自に停止します。ただし、価格改定・外部課金・処理の遅延などを含む完全な請求保証ではありません。OpenAI側でもプリペイド、自動チャージOFF、予算通知を併用してください。

## 必要な環境変数・シークレット

デプロイ先のサーバー側に設定します。`.env` やシークレット管理機能を使い、GitHubへコミットしないでください。

```text
OPENAI_API_KEY=sk-proj-...

# 推奨：公開URLを知った第三者の利用を防ぐクラス用合言葉
APP_ACCESS_CODE=任意の合言葉

# 推奨初期値。未設定でも下記の値が使われます
AI_MONTHLY_BUDGET_USD=1
AI_MONTHLY_REQUEST_LIMIT=2000
AI_PER_MINUTE_LIMIT=2

# 任意：レート制限用ハッシュを環境ごとに変える秘密文字列
AI_RATE_LIMIT_SALT=十分長いランダム文字列
```

`OPENAI_MODEL` は使いません。誤って高額なモデルに切り替わらないよう、コード内で `gpt-5.4-nano` に固定しています。

## 月額上限の動き

日本時間の月ごとに、次のどちらかへ達した時点でAI精査を停止します。

1. `AI_MONTHLY_BUDGET_USD` の概算利用額
2. `AI_MONTHLY_REQUEST_LIMIT` の成功リクエスト数

標準値は月1ドル・月2,000回です。AI精査画面には当月の回数と概算額を表示します。上限到達後も無料ルーブリック判定は使えます。

予算判定では、GPT-5.4 nanoの標準処理料金として次をコード内で使用しています。

```text
入力：$0.20 / 100万トークン
キャッシュ入力：$0.02 / 100万トークン
出力：$1.25 / 100万トークン
```

料金が改定された場合は、`app/api/evaluate/route.ts` 冒頭の料金定数を更新してください。

## D1データベース

`.openai/hosting.json` の `d1` は `DB` に設定済みです。アプリは起動時に必要なテーブルを作成します。

保存するテーブル：

- `ai_monthly_usage`：月ごとの回数・トークン・概算料金
- `ai_rate_limits`：短時間の連続送信を防ぐハッシュ化識別子

児童の入力本文や評価結果は保存しません。

## 開発

Node.js 22.13以上を使用します。

```bash
npm ci
npm run dev
npm run lint
npm test
```

APIをローカルで試す場合は、Cloudflare/Viteのローカル環境へD1とシークレットを設定してください。API未設定時でも画面と無料ルーブリック判定は動作します。

## 運用前の確認

- OpenAI Platformでプリペイドを使い、Auto rechargeをOFFにする
- OpenAI側のプロジェクト予算通知も設定する
- APIキーはこのアプリ専用プロジェクトで発行する
- APIキーをHTML、JavaScript、GitHubへ直接書かない
- `APP_ACCESS_CODE` を設定して児童だけに伝える
- 児童名、出席番号、学校名を入力させない
- AI・自動判定だけで評定や指導要録を決定しない
- 学校・自治体の情報セキュリティ方針を優先する
